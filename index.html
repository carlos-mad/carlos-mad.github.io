Prácticas de Linux

1. Navegación por el sistema de archivos

Comandos:

  • pwd - Mostrar el directorio actual
  • ls - Listar contenido del directorio
  • cd /ruta - Cambiar de directorio

Ejercicio: Accede al directorio /etc y lista sus archivos.

2. Gestión de archivos y directorios

  • touch archivo.txt - Crear un archivo vacío
  • mkdir carpeta - Crear un directorio
  • rm archivo.txt - Eliminar un archivo

Ejercicio: Crea una carpeta llamada practicas y dentro un archivo llamado info.txt.

3. Permisos de archivos

  • chmod - Cambiar permisos
  • chown - Cambiar propietario

Ejercicio: Cambia los permisos del archivo info.txt para que sólo tú puedas leerlo y escribirlo.
